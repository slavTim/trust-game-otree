# Trust Game (Игра в доверие)

Экспериментальная экономическая игра, разработанная на фреймворке oTree.

## Описание

Игра демонстрирует механизмы доверия и взаимности в экономическом взаимодействии. Участники играют в парах, где один выступает в роли Отправителя (Sender), другой — Получателя (Receiver). Каждый раунд пары и роли случайным образом перетасовываются.

## Правила игры

1. Отправитель получает 100 очков.
2. Отправитель решает, сколько очков отправить Получателю (от 0 до 100).
3. Отправленная сумма утраивается (умножается на 3).
4. Получатель решает, сколько очков вернуть Отправителю.
5. Выигрыши рассчитываются автоматически.

Формулы расчёта:

- Выигрыш Отправителя = 100 - Отправлено + Возвращено
- Выигрыш Получателя = (Отправлено * 3) - Возвращено

## Ключевые особенности

- Перетасовка групп и ролей в каждом раунде
- Асинхронный режим (группы не ждут друг друга)
- Показ общего выигрыша за все раунды
- Проверка: Получатель не может вернуть больше полученного
- Интерфейс на русском языке

## Настройка параметров

Параметры игры находятся в файле trust/__init__.py:

- num_rounds - количество раундов (по умолчанию 5)
- endowment - начальная сумма (по умолчанию 100)
- multiplication_factor - множитель отправленной суммы (по умолчанию 3)

## Установка и запуск

Требования: Python 3.12 или выше

Клонирование репозитория:
git clone https://github.com/slavTim/trust-game-otree.git
cd trust-game-otree

Установка oTree:
pip install otree

Запуск сервера:
otree devserver

После запуска откройте в браузере http://localhost:8000
